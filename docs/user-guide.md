# 使用方法(部員向けマニュアル)

対象URL: https://tennis-app-theta-lemon.vercel.app

## ログイン

1. 上記URLにアクセスすると、ログイン画面が表示されます
2. 「Googleでログイン」ボタンを押し、部で登録されているご自身のGoogleアカウントでログインしてください
3. 「このGoogleアカウントは部の許可リストに登録されていません」と表示された場合は、そのGoogleアカウントがまだ許可されていません。管理者に連絡してメールアドレスを登録してもらってください

**注意**: 必ずSafariやChromeなどの通常のブラウザで開いてください。Google検索アプリなどアプリ内蔵のブラウザから開くと、Google側の制限によりログインできません。

## 画面の見方

ログインすると以下の3つのエリアが表示されます。

- **左側「フォルダ」**: フォルダの階層構造。タップ/クリックで中のファイルを絞り込み表示します。「すべて」を選ぶと全フォルダのファイルが表示されます
- **上部「タグ」**: タグのチップをタップすると、そのタグが付いたファイルだけに絞り込まれます。検索窓にキーワードを入力すると、一致するタグだけが表示されます
- **メイン「ファイル一覧」**: ファイル名・サイズ・タグ・操作ボタンが並びます

## ファイルの閲覧・ダウンロード

各ファイルの右側に2つのボタンがあります。

- **表示**: 新しいタブでファイルを開きます(PDFなどはそのままブラウザ内で見られます)
- **DL**: お使いの端末に保存します(保存先の選択やダウンロードフォルダへの保存が行われます)

## タグを付ける

各ファイルの「タグ」欄にある入力欄に、タグ名を入力してEnterキーを押すと付与されます。

- **カンマ区切りで複数同時に入力可能**です(例: `2026年度,大会,倉敷市大会`)
- 既に同じ名前のタグがあれば、それに付与されます(重複作成はされません、大文字小文字は区別しません)
- タグの右側の「×」で取り外せます(自分が付けたタグ、または管理者が付けたタグを外せます)

## できないこと(一般部員)

以下は管理者のみが行える操作です。表示されていても押せない、または押しても失敗します。

- フォルダの作成・名前変更・削除
- ファイルのアップロード・削除
- ログイン許可メールアドレスの追加・削除

これらが必要な場合は管理者に依頼してください。

## うまく動かないとき

- **ボタンを押しても反応しない**: ページを再読み込みしてから試してください
- **ダウンロードが始まらない**: ブラウザのポップアップブロックが原因の可能性があります。設定でこのサイトのポップアップを許可してください
- **ログインでエラーが出る**: メールアドレスが許可リストに登録されているか、管理者に確認してください
